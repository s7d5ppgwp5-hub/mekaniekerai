# Straddler Carrier AI v3.0

Technical Intelligence System voor Straddler Carrier onderhoud en beheer.

## Installatie

### Lokale Ontwikkeling
1. Clone deze repository
2. Open `index.html` in je browser
3. Of gebruik: `python -m http.server 8080`

### Vercel Deployment
1. Upload alle bestanden naar GitHub
2. Verbind je repository met Vercel
3. Vercel herkent automatisch de `vercel.json` configuratie

## Configuratie

### API Sleutel
Voor AI functionaliteit heb je een Anthropic API sleutel nodig:
1. Ga naar [console.anthropic.com](https://console.anthropic.com)
2. Maak een API sleutel
3. Gebruik de 🔑 knop in de app om de sleutel in te stellen
